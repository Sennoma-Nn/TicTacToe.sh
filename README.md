# TicTacToe.sh

A simple Tic-Tac-Toe game written in Zsh script for terminal.

## About

This is a practice project - a terminal-based Tic-Tac-Toe game that supports mouse interaction.

## Features

- Mouse click support
- Terminal graphics using Unicode characters
- Complete game logic (win/draw detection)
- Single script file, no dependencies

## Usage

```bash
./TicTacToe.sh
```

Click on the board to place X and O. Press Ctrl+C to exit.
